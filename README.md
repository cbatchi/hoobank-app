## 🚀 Quick Start

### Prerequisites
- **Node.js** 16+ 
- **npm** 8+ or **yarn** 1.22+

### Installation

```bash
# Clone the repository
git clone https://github.com/cbatchi/hoobank-app.git

# Navigate to project
cd hoobank-app

# Install dependencies
npm install

# Start the development server
npm run dev
```

That's it! Your Hoobank experience starts now 🎉
